# タイトル文字修飾プラグイン（NLM_TitleFont.js）
### RPGツクールMZ/MV両用プラグイン

タイトル文字を簡易的に色々修飾できます  
　サイズ、カラー、縁取り、太字、斜字、フォントファイル、影付き文字、背面帯、サブタイトルなどを パラメータで変えられます
